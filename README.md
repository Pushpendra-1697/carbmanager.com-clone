## Run Locally

Clone the project

```bash
  git clone https://github.com/abhizeit/kaput-chin-6583.git
```

Go to the project directory

```bash
  cd animoto
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
